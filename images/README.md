# 이미지

이 폴더에 실제 이미지 파일을 넣으면 빌드 시 자동으로 사용되고, 없으면 placeholder가 렌더링됩니다.

## 홈에서 사용되는 이미지

| 파일 | 용도 | 권장 규격 |
|------|------|-----------|
| `hero.jpg` | 히어로 좌측 이미지 | 세로 4:5 (예: 960 × 1200) |
| `value-namu.jpg` | 나무(어르신) 가치 카드 | 가로 4:3 (예: 800 × 600) |
| `value-gaji.jpg` | 나뭇가지(가족) 가치 카드 | 가로 4:3 (예: 800 × 600) |

## 블로그 썸네일

블로그 글 frontmatter에 `thumbnail: filename.jpg`를 명시하면 이 폴더에서 찾습니다. 규격은 16:10 (예: 1280 × 800) 권장. 미설정 시 카테고리 이모지 placeholder가 사용됩니다.

## 포맷

- JPG (사진) 또는 WebP 권장
- 파일당 300KB 이하 목표
- 원본 EXIF은 제거하시고 sRGB로 저장해주세요
